# Rust notes

## Installation
```console
apt update && apt install -y curl && \
curl https://sh.rustup.rs -sSf | sh -s -- -y
source $HOME/.cargo/env
```

## Compile
```console
rustc main.rs
```

## cargo
```console
cargo new projectX
cargo build

# build and run
cargo run
```
